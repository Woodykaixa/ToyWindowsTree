# ToyWindowsTree

A toy level program to list contents of directories in a tree-like format. 
This program use {fmt} to format strings.

## Compilation

Clone [{fmt}](https://github.com/fmtlib/fmt) into `/lib/fmt`. Then compile with CMake.
